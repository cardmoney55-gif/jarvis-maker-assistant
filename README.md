# J.A.R.V.I.S. — Автономний асистент майстра

Власний AI-асистент у стилі «Залізної людини» для збірки та пайки електроніки.
Розмовляє **українською**, **бачить** через камеру, веде **живу голосову розмову**, **сам шукає в інтернеті**, **накопичує знання**, малює **3D-голограми деталей** і виконує **навички**. Працює як **десктоп-програма** (Electron), не в браузері.

---

## Можливості

| Можливість | Опис |
|---|---|
| 🧠 Розум | Google **Gemini** (хмара) у ролі «мозку». Модель легко змінити через env. |
| 🔍 Власний веб-пошук | JARVIS сам гуглить (DuckDuckGo) і читає сторінки — **без квоти Google, безкоштовно** (`websearch.ts`). |
| 💾 Памʼять, що навчається | Векторна памʼять (RAG, `memory.ts`): згадує знання перед відповіддю й зберігає нові. Переживає перезапуск. |
| 📷 Зір | Розпізнавання компонентів і пінів з камери/фото; розпізнані деталі **самі додаються у схему пайки**. |
| 🔮 **3D-голограми** | Обертові голографічні моделі деталей **із підписами пінів** (Three.js). Скажи «покажи діод» — і модель зʼявиться у вільній області, крути її мишею. |
| 🧩 Навички | Розширювана система здібностей (Gemini function-calling, `skills.ts`): закон Ома, резистор для LED, кольори резистора, 3D-візуалізація, запамʼятовування. Нову навичку додаєш одним записом. |
| 🎙 Живий режим | Постійне слухання без кнопки + визначення кінця фрази (VAD) + **переривання** (почав говорити — JARVIS замовкає й слухає). Як у фільмі. |
| 🗣 Нейроголос | **Piper** (локальний, безкоштовний, українська) або **XTTS**-клон голосу; фолбек на системний голос (`tts.ts`). |
| 🖥 Десктоп | Власне вікно (Electron), збірка у `.exe`. |

## Запуск (розробка)

**Потрібно:** Node.js 18+

1. Встановити залежності:
   ```bash
   npm install
   ```
2. Створити файл `.env` і вписати безкоштовний ключ Gemini (https://aistudio.google.com/apikey):
   ```
   GEMINI_API_KEY="ваш_ключ"
   ```
3. Запустити як **десктоп-програму**:
   ```bash
   npm run electron:dev
   ```
   (або як веб-сервер для розробки: `npm run dev` → http://localhost:3000)

## Збірка `.exe`

```bash
npm run dist:win
```
Результат у папці `release/`:
- `JARVIS-Maker-Assistant-*-x64.exe` — інсталятор
- `JARVIS-Maker-Assistant-*-portable.exe` — портативна версія

Для роботи AI поклади файл `.env` з `GEMINI_API_KEY` поруч із встановленим `.exe`.

## Налаштування (env)

| Змінна | Призначення | Типове значення |
|---|---|---|
| `GEMINI_API_KEY` | Ключ Gemini (обовʼязково) | — |
| `JARVIS_GEN_MODEL` | Модель «мозку» | `gemini-3.5-flash` |
| `JARVIS_EMBED_MODEL` | Модель векторів памʼяті | `gemini-embedding-001` |
| `JARVIS_MEMORY_PATH` | Файл памʼяті (можна вказати папку Google Drive) | `./jarvis-memory.json` |
| `JARVIS_PIPER_PATH` / `JARVIS_PIPER_MODEL` | Нейроголос Piper (бінарник + .onnx голос) | — (без них — системний голос) |
| `JARVIS_TTS` / `JARVIS_XTTS_URL` | Клон-голос через XTTS-сервер | `piper` |
| `PORT` | Порт бекенду | `3000` |

### Нейроголос (Piper) — безкоштовно

За замовчуванням JARVIS говорить системним голосом. Для природного нейроголосу:
1. Завантаж Piper і голос (uk_UA або en_GB) з [piper releases](https://github.com/rhasspy/piper/releases) і [piper-voices](https://huggingface.co/rhasspy/piper-voices).
2. У `.env` вкажи `JARVIS_PIPER_PATH` та `JARVIS_PIPER_MODEL`.
3. Перезапусти — JARVIS автоматично перейде на нейроголос (бекенд віддає аудіо, клієнт його програє).

Для клонування голосу (напр. як у фільмі) підніми XTTS-сервер і встанови `JARVIS_TTS=xtts` + `JARVIS_XTTS_URL`.

## Архітектура

```
electron/main.cjs      запускає бекенд + відкриває вікно
server.ts              Express API: чат, зір, пошук деталей, навчання
memory.ts              векторна памʼять (RAG) — самонавчання
websearch.ts           власний безкоштовний веб-пошук (DuckDuckGo + reader)
src/                   React UI (ядро, камера, схема пайки, памʼять, чат)
```

> **Примітка про «розум»:** на цьому залізі (без GPU) локальні моделі неможливі, тому мозок — хмарний Gemini, який легко замінити. Уся система навколо (агент, памʼять, веб-пошук, зір, голос) — власна розробка.
